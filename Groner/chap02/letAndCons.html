Let e Const - declarações de variáveis em Javascript

Javascript e conhecida por ser um linguagem fracamente tipada na hora de declarações de variáveis. diferentemente de C/C++ ou Java que são consideradas linguagens fortemente tipada.

É considerada de tipagem fraca porque não precisamos declarar qual tipo primitivo é a variável, só durante a execução do programa que fica mais claro que tipo é a variável.

Let, Const e Object - aplicação simples

Escopo das variaveis

Uma variável tem um escopo definido, dependendo onde forem declaradas as variáveis. Se for declarada dentro de um bloco ela pertence apenas ao bloco, sendo declarada dentro de uma função, ela pertence apenas a função, podendo ser usada em qualquer lugar dentro da função. Se for declarada no programa, ela pode ser usada em todo o programa.
