True and False

Exemplos de aplicação de Verdadeiro ou Falso em Javascript

Para ver os resultados do script em javascript abra a janela do Console do seu nagavador.

undefined - false

null - false

Boolean - Verdeiro é true e falso é false

Number - O resultado é fale para +0, -0 e NaN; caso contrário é true

String - O resultado é falso se a string for vazia (o tamanhp é 0); caso contrário, é true (tamanho >=1)

Object - true
