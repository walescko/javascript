Tipos de Variáveis

O Typescrispt trás tipos de variáveis assim como outras linguagens de programação, porém ainda trás alguns tipos a mais de variáveis que outras linguagens não possuem.

  • Boolean: True or False
  • String: são caractetizadas pelo uso de aspas simples(''), aspas duplas ("") ou backstrik (``); particularmente eu prefiro as aspas duplas.
  • number: para uso de números de qualquer tipo: int, float, hexadecima, binary, etc..
  • Array: declaradado com [], porém deve-se informar o tipo de variável: string[] - array de strings, number[], array de números; pode-se declarar um array também da seguinte forma: meuArray = array<type>
